# Studenthjul og Spørsmålshjul

Et enkelt verktøy for aktiv læring i undervisning.

Dekker pedagogisk teknikk "No Hands" 
Læerings strategi "Bevis for læring"

Verktøyet kan brukes til:

- Tilfeldig valg av student
- Tilfeldig valg av spørsmål
- Muntlige aktiviteter i klasserommet
- Seminarer og gruppearbeid
- Nett- og campusundervisning

## Funksjoner

- Studenthjul
- Spørsmålshjul
- Støtte for flere klasser
- Støtte for flere emner og moduler
- Import av studenter, spørsmål og svar
